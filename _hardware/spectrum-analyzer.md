## Overview

A portable RF spectrum analyzer for learning radio frequency fundamentals. Built around the ESP32 with an OLED display for real-time visualization.

## Goals

- Understand RF signal characteristics and behavior
- Learn spectrum analysis fundamentals
- Build a portable tool for debugging RF projects

## Hardware

| Component | Description |
|-----------|-------------|
| ESP32 | Main microcontroller |
| OLED Display | Real-time spectrum visualization |
| Antenna + Pigtail | Signal reception |
| SD Module | Data logging |
